parseArgs

public ParseResult parseArgs(String argString)

Parse a string into a list of strings, similar to how a shell interpreter would split it. Then parse those args pieces into a ParseResult. Pieces are split on whitespace, and literal strings starting with either single or double quotes may encapsulate any characters until a closing quotation is found. Within those literal strings, a quote may be escaped with a backslash character.
